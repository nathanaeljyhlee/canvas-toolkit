# Canvas Toolkit - Quick Start Guide

**Get your Canvas content (assignments, announcements, modules) in Excel with 3 simple steps.**

No coding required. Takes 5 minutes.

---

## Step 1: Install Python (One-Time Setup)

**Download Python:**
1. Go to [python.org/downloads](https://www.python.org/downloads/)
2. Click the big yellow "Download Python" button
3. Run the installer

**IMPORTANT:** Check the box that says **"Add Python to PATH"** ‚úÖ

![Python Install](https://docs.python.org/3/_images/win_installer.png)

**Click "Install Now"** and wait ~2 minutes.

**Verify it worked:**
- Open Command Prompt (search "cmd" in Windows)
- Type: `python --version`
- You should see: `Python 3.x.x`

---

## Step 2: Install Canvas Toolkit (One Command)

**Download the installer:**
1. Go to https://github.com/nathanaeljyhlee/canvas-toolkit
2. Click the green **"Code"** button
3. Click **"Download ZIP"**
4. Extract the ZIP file
5. Find `install-canvas-toolkit.bat`

**Run the installer:**
- Double-click `install-canvas-toolkit.bat`
- Wait 1-2 minutes while it installs
- You'll see "Installation successful!"

**That's it!** Canvas Toolkit is now installed.

---

## Step 3: Run Canvas Toolkit

**Launch the app:**
1. Double-click `run-canvas-toolkit.bat`
2. Your browser will open to `http://localhost:8501`
3. Keep the black window (terminal) open while using the app

**Get your Canvas API token:**
1. Log into Canvas ‚Üí Account ‚Üí Settings
2. Scroll to "Approved Integrations"
3. Click "+ New Access Token"
4. Enter purpose: "Canvas Toolkit"
5. Copy the token (shown only once!)

**Export your content:**
1. Paste your API token in the app
2. Select which courses you want
3. **Choose content types** (Assignments ‚úì, Announcements, Modules)
4. Choose export format (Excel recommended)
5. Click "Export Content"
6. Download your file!

---

## Tips

**To stop the app:**
- Close the browser tab
- Close the black terminal window

**To run it again:**
- Just double-click `run-canvas-toolkit.bat`

**To update to a new version:**
- Run `install-canvas-toolkit.bat` again
- It will update automatically

**Having issues?**
- Make sure Python is installed (`python --version`)
- Try running Command Prompt as Administrator
- Check [SETUP.md](SETUP.md) for detailed troubleshooting
